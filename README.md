# the-weighter-backend
실행 방법
- npm start
- nodemon app

폴더 구조
- src : 사실상 본체
  - config : Sequelize에 대한 config 폴더
  - controller : 실제로 Sequelize가 동작하는 컨트롤러들이 있는 폴더
  - models : Sequelize에 사용하기 위한 테이블에 대한 정보가 있는 폴더
  - router : controller이 작동하는 주소, 각 주소에 대한 controller 부여되어 있는 폴더

init.js
- 포트 설정

server.js
- 기존의 app.js 역할
