# the-weighter-backend
実行方法
- npm start
- nodemon app

フォルダー構造
- src
  - config : Sequelizeに対するconfigフォルダ
  - controller : 実際にSequelizeが動作するコントローラがあるフォルダ
  - models : Sequelizeに使用するためのテーブルに関する情報があるフォルダ
  - router : controllerが動作するアドレス、各アドレスに対するcontrollerが付与されているフォルダ

init.js
- ポート設定

server.js
- 既存のapp.js役割
